# Setup for ssh environment

### Install package in ~/Downloads

### Install deb files

- nvim
- rg

```bash
wget “link”
dpkg -x path-to-deb ~/
```

### Install tar.gz files

- lazygit
- stow-2.3.1
- tmux-3.4
- starship

```bash
wget “link”
tar zxvf path-to-tar.gz

# read installation markdown file if didn't give bin
```
